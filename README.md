PharmaHeaven Telegram Bot

Telegram-бот PharmaHeaven — это каталог продукции с FAQ и системой заявок на консультацию.
Бот не осуществляет продажи и не принимает оплату — используется как информационная витрина и инструмент связи с менеджером.

⸻

🚀 Возможности
	•	📦 Каталог товаров
	•	📄 Детальные карточки продуктов
	•	❓ Раздел FAQ
	•	⚠️ Информационный дисклеймер
	•	📝 Сбор заявок от пользователей
	•	💾 Сохранение заявок в SQLite
	•	🔔 Уведомления админу в Telegram

⸻

📁 Структура проекта

pharmaheaven_bot_ru.py   # основной файл бота
pharmaheaven.db         # база данных (создаётся автоматически)


⸻

⚙️ Установка и запуск

1. Клонировать репозиторий

git clone https://github.com/yourusername/pharmaheaven-bot.git
cd pharmaheaven-bot

2. Установить зависимости

pip install python-telegram-bot==22.*

3. Настроить переменные окружения

export BOT_TOKEN="ВАШ_ТОКЕН_БОТА"
export ADMIN_CHAT_ID="ВАШ_CHAT_ID"
export MANAGER_CONTACT="@your_manager"

4. Запустить бота

python pharmaheaven_bot_ru.py


⸻

🛠 Настройка

📦 Добавление товаров

Открой файл pharmaheaven_bot_ru.py и найди блок:

PRODUCTS = [
    {
        "name": "Название",
        "description": "Описание",
        "price": "Цена"
    }
]

Добавь свои товары в этот список.

⸻

💬 Изменение FAQ

Отредактируй список:

FAQ = [
    ("Вопрос", "Ответ"),
]

🧠 Как работает заявка

Пользователь проходит шаги:
	1.	Имя
	2.	Возраст
	3.	Цель
	4.	Опыт
	5.	Контакт

После этого:
	•	заявка сохраняется в базу
	•	админу приходит уведомление
